# BMI Calculator 💪

A simple and efficient Body Mass Index (BMI) calculator application. This tool allows users to easily calculate their BMI by inputting their height and weight, and provides immediate feedback on their weight category based on standard health guidelines.

## 🚀 Features

* **Quick Calculation:** Instantly computes BMI using standard formulas.
* **Health Categorization:** Classifies the result into standard categories (Underweight, Normal weight, Overweight, Obesity).
* **User-Friendly:** Clean and intuitive interface for seamless user experience.
* **Input Validation:** Prevents incorrect data entries (e.g., negative numbers or text).
